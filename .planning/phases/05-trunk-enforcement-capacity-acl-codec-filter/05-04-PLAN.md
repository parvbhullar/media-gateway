Promote Phase 5 schema into proxy hot-path enforcement. Three gates land in
`match_invite_with_trace`, in this exact sequence (after global firewall, after the
existing trunk resolution step):

1. **Per-trunk ACL** (D-15, D-16) — DB read of `supersip_trunk_acl_entries`, evaluated
   top-to-bottom by position; first match wins; default = allow (D-14). Deny → 403.
2. **Capacity gate** (D-03) — atomic `try_acquire` on the in-memory `TrunkCapacityState`;
   checks `max_calls` and consumes 1 token from the CPS token bucket. Failure → 503 +
   `Retry-After: 5`.
3. **Codec intersection** (D-18, TSUB-06) — caller SDP audio codec list ∩
   `media_config.codecs`; empty → 488.

Plus the observability wiring (D-02, D-04, TSUB-07): GET /capacity returns live
`current_active` (registry filtered by `trunk_group_name`) and `current_cps_rate`
(token bucket snapshot), replacing the Plan 05-02 placeholders.

Plus IT-03 integration tests in `tests/proxy_trunk_enforcement.rs` driving each gate
through the dispatch flow.

**SDP / caller-codecs locked decision:** `match_invite_with_trace` accepts a new
`caller_codecs: Vec<String>` parameter. The caller (the matcher's callsite —
`src/proxy/call.rs` and any other invocation site) is responsible for parsing SDP and
passing the audio codec name list. This keeps the matcher pure and SDP parsing at the
appropriate layer (call.rs already handles SDP). See Task 4 sub-task 4d.

Token-bucket sketch (D-06, D-07 — hand-rolled, no new dep):
```rust
pub struct TrunkCapacityGate {
    active: AtomicU32,
    max_calls: AtomicU32,        // 0 = unlimited
    bucket_tokens: AtomicU32,    // current available tokens
    bucket_max: AtomicU32,       // max_cps; 0 = unlimited
    bucket_last_refill_ms: AtomicU64,  // epoch ms of last refill calculation
}
impl TrunkCapacityGate {
    pub fn try_acquire(&self) -> AcquireOutcome { /* refill, then atomic CAS check both */ }
    pub fn release(&self) { self.active.fetch_sub(1, Relaxed); }
    pub fn snapshot_cps_rate(&self) -> u32 { /* tokens consumed in the last full second */ }
    pub fn update_limits(&self, max_calls: Option<u32>, max_cps: Option<u32>) { ... }
}
pub enum AcquireOutcome {
    Ok(Permit),
    CallsExhausted,   // → 503 + reason trunk_capacity_exhausted
    CpsExhausted,     // → 503 + reason trunk_cps_exhausted
}
pub struct Permit {
    gate: Arc<TrunkCapacityGate>,
}
impl Drop for Permit { fn drop(&mut self) { self.gate.release(); } }
```
