**Path resolution algorithm:**

```rust
fn resolve_recording_path(
    explicit: Option<&str>,
    session_id: &str,
    format: &str,          // "wav" or "mp3" (post-validation)
    recorder_path: &str,
) -> ApiResult<String> {
    use std::path::{Path, PathBuf};
    let recorder_root = PathBuf::from(recorder_path);
    let ext = format;  // format already validated to wav/mp3

    match explicit {
        None => {
            // Auto-generate: <recorder_path>/<session_id>-<unix_ts>.<ext>
            let ts = chrono::Utc::now().timestamp();
            let filename = format!("{}-{}.{}", session_id, ts, ext);
            Ok(recorder_root.join(filename).to_string_lossy().to_string())
        }
        Some(raw) => {
            if raw.is_empty() {
                return Err(ApiError::bad_request("empty recording path"));
            }
            if raw.contains("..") {
                return Err(ApiError::bad_request(
                    "recording path must not contain '..'",
                ));
            }
            let p = Path::new(raw);
            if !p.is_absolute() {
                return Err(ApiError::bad_request(
                    "recording path must be absolute",
                ));
            }
            // Must be inside the recorder tree (string-prefix check is
            // sufficient because we already rejected '..').
            if !p.starts_with(&recorder_root) {
                return Err(ApiError::bad_request(format!(
                    "recording path must be inside recorder tree '{}'",
                    recorder_root.display()
                )));
            }
            Ok(raw.to_string())
        }
    }
}
```

```rust
fn maybe_drop_transcribe_marker(path: &str, transcribe: bool) -> ApiResult<()> {
    if !transcribe { return Ok(()); }
    let marker = format!("{}.transcribe.marker", path);
    // Best-effort; failures here are NOT fatal to the recording dispatch
    // (the marker is a hint for Phase 7+ webhooks). Log on failure but
    // do not fail the request.
    match std::fs::File::create(&marker) {
        Ok(_) => Ok(()),
        Err(e) => {
            tracing::warn!("failed to create transcribe marker '{}': {}", marker, e);
            // Return Ok so the operator still gets 200 on the record dispatch.
            // An alternative is to fail the request with 500, but marker
            // failure shouldn't block the call from being recorded.
            Ok(())
        }
    }
}
```

```rust
    explicit: Option<&str>,
    session_id: &str,
    format: &str,
    recorder_path: &str,

    use std::path::{Path, PathBuf};

            return Err(ApiError::bad_request(

/// Drop an empty marker file alongside the recording path when
/// `transcribe:true`. Phase 7 (Webhooks) consumers check for this sibling
/// when they process `callrecord/` completion events.
///
/// Marker creation is best-effort — failures are logged but do not block
/// the record dispatch. The marker is a HINT, not a contract — operators
/// can tolerate missing markers and retry by re-calling /record if needed.
fn maybe_drop_transcribe_marker(path: &str, transcribe: bool) {
    if !transcribe {
        return;
    }
    let marker = format!("{}.transcribe.marker", path);
    match std::fs::File::create(&marker) {
        Ok(_) => {
            tracing::info!("dropped transcribe marker: {}", marker);
        }
        Err(e) => {
            tracing::warn!(
                "failed to drop transcribe marker '{}': {}; continuing",
                marker, e
            );
        }
    }
}
```

```rust
async fn record_call(
    State(state): State<AppState>,
    Path(id): Path<String>,
    Json(req): Json<RecordRequest>,
) -> ApiResult<Json<serde_json::Value>> {
    let registry = state.sip_server().inner.active_call_registry.clone();
    let _ = require_session(&registry, &id)?;

    // D-17: validate format up-front.
    let format = validate_recording_format(req.format.as_deref())?;

    // D-16: resolve path (auto-generate or validate explicit).
    let recorder_path = state.config().recorder_path();
    let resolved = resolve_recording_path(
        req.path.as_deref(),
        &id,
        &format,
        &recorder_path,
    )?;

    // D-18: drop transcribe marker BEFORE dispatch so Phase 7 observers can
    // detect it even if dispatch fails. Best-effort (see helper doc).
    maybe_drop_transcribe_marker(&resolved, req.transcribe.unwrap_or(false));

    let payload = CallCommandPayload::Record {
        path: Some(resolved.clone()),
        format: Some(format),
        beep: req.beep,
        max_duration_secs: req.max_duration_secs,
        transcribe: req.transcribe,
    };
    let extra = Some(serde_json::json!({ "path": resolved }));
    map_command_result(dispatch_console_command(&registry, &id, payload), extra)
}
```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-05-01 | Spoofing | unauthenticated /record | mitigate | Bearer auth inherited; `record_requires_auth` asserts 401. |
| T-04-05-02 | Tampering | path traversal via `..` | mitigate | `resolve_recording_path` rejects any path containing `..`; `record_path_traversal_returns_400` test asserts 400. |
| T-04-05-03 | Tampering | path escape via symlink inside recorder tree | accept (defense-in-depth) | Current impl uses string-prefix `starts_with(recorder_root)` without `canonicalize`. A symlink inside the recorder dir pointing outside would escape. Mitigation deferred to v2.1 hardening — documented in this plan's security consideration. Production deployments should keep the recorder dir free of unverified symlinks. |
| T-04-05-04 | Tampering | absolute path outside recorder tree | mitigate | `starts_with(recorder_root)` check rejects any path not under the configured recorder dir; `record_outside_tree_returns_400` test asserts 400. |
| T-04-05-05 | Tampering | relative path bypassing root check | mitigate | `p.is_absolute()` check rejects relative paths up-front; `record_relative_path_returns_400` test asserts 400. |
